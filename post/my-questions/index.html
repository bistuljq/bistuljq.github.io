My questions

2021-10-15 1 min read

老师好，我是软工2002班的李佳奇。在这次创建博客的过程中，我遇到的配置的问题有两个。
1.我在使用Gridea软件来在Github网站搭建博客的过程中，遇到了不能同步的问题。
在查阅资料后，我知道了问题出在系统路径中没有我的用户名和邮箱信息。因此我在output文件夹，使用git命令自行添加邮箱地址和用户名，代码如下：

配置邮箱

git config --global user.email "3055850273@qq.com"
配置名称

git config --global user.name "bistuljq"
从而解决了我的问题。

2.我在GitHub网站中创建存储器时也遇到了问题，那就是由于我的用户名中有大写字母，创建时失败。之后我重新创建了账号，使用小写字母，解决了这个问题。
